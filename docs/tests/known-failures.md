## Severity guide

| Level | Meaning |
| --- | --- |
| **S0** | Crash / data loss / soft-lock |
| **S1** | Incorrect orders / broken commander loop with default config |
| **S2** | Visual / feedback glitch; debug-only wrong text |
| **S3** | Doc / UX polish |

## How to add a row

1. Assign **KF-NNN** (increment NNN).
2. Link GitHub issue or internal ticket in **Description** if filed.
3. Prefer **exact config keys** and **custom battle setup** in reproduction.
4. When fixed, move row to **Resolved** (append table below) with version fixed-in.
