# Portfolio evidence pack — RTS Commander Doctrine

This folder is a **documentation-only** portfolio scaffold for **Bannerlord.RTSCameraLite** (RTS Commander Doctrine mod for *Mount & Blade II: Bannerlord*). It is meant to pair **recorded evidence** (clips, screenshots, diagrams) under `media/` with **written narrative** recruiters and hiring managers can skim in under a few minutes.

## How it is organized

| File | Purpose |
|------|---------|
| [90-second-demo-script.md](90-second-demo-script.md) | Shot list and narration beats for a single short capture pass. |
| [feature-proof-checklist.md](feature-proof-checklist.md) | What to prove on camera or in logs for each slice-level capability. |
| [before-after-comparison-plan.md](before-after-comparison-plan.md) | Native game vs mod framing (honest scope — not every row is “shipped”). |
| [recruiter-summary.md](recruiter-summary.md) | One-screen pitch: skills, systems, constraints, target roles. |
| [technical-breakdown.md](technical-breakdown.md) | Architecture vocabulary aligned to this repo (MissionView, adapters, doctrine, commands). |
| [design-case-study-outline.md](design-case-study-outline.md) | Long-form case study skeleton (problem → approach → tradeoffs → results). |
| [resume-bullets.md](resume-bullets.md) | Role-tuned bullet variants. |
| [linkedin-project-post.md](linkedin-project-post.md) | Public, portfolio-safe WIP announcement. |

## Media

Captured assets live under **`media/`** at the repository root (see `media/README.md`). Naming conventions keep evidence sortable by date and feature.

## Truth rules

- Prefer **“implemented in repo per slice doc”** over marketing superlatives.
- Items **hard-gated** on research or engine verification are called out explicitly (see `docs/slice-hard-gates.md` in the repo).
- The high-level roadmap in `docs/slice-roadmap.md` may **lag** the codebase; the slice markdown under `docs/slices/` is the first-line technical reference.

## Slice 22 scope

This slice adds **docs and media planning structure only** — no gameplay code.
