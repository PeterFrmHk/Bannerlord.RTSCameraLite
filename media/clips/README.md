# Clips

## Naming

`YYYY-MM-DD_feature-name.mp4`

Examples:

- `2026-04-18_rts-commander-90s-demo.mp4` — full scripted pass (`docs/portfolio/90-second-demo-script.md`).
- `2026-04-18_cavalry-doctrine-states.mp4` — log-visible state transitions.

## Capture notes

- Record at a stable resolution (1080p minimum for portfolio reuse).
- Leave **1s handles** at head/tail for editing.
- Prefer **clean UI** or **log window** visible; avoid copyrighted music in the capture track.
