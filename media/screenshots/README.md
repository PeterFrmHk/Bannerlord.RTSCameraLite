# Screenshots

## Naming

`YYYY-MM-DD_feature-name.png`

Examples:

- `2026-04-18_commander-mode-toggle.png` — Backspace / config key visible with mode change evidence.
- `2026-04-19_marker-fallback-text.png` — throttled `[Marker]` or tactical feedback line.

## Tips

- Crop to the **minimum readable** region; add a thin border if posting on light backgrounds.
- When capturing logs, **increase font size** before screenshot for readability.
